> **연규님께:** 기술적인 빌드는 이미 끝났습니다. 연규님은 GitHub Releases 페이지에서 파일 하나만 다운로드하면 됩니다. 아래 1번 섹션부터 읽어보세요.

## 2. 처음 받기 — GitHub Releases 페이지

### GitHub Releases 페이지 찾는 법

GitHub Releases를 처음 사용하는 경우 아래 순서를 따라가세요.

1. GitHub에서 `obsidian-ai-assistant` 저장소 페이지를 엽니다.
2. 오른쪽 사이드바를 보면 **"Releases"** 항목이 있습니다. 클릭합니다.
3. 가장 위에 있는 버전(예: `v3.0.0`)이 최신 버전입니다.
4. 해당 버전을 클릭하면 상세 페이지가 열립니다.
5. 페이지 아래쪽 **"Assets"** 섹션을 클릭해 펼칩니다.
6. 아래 표를 참고해 본인 OS에 맞는 파일을 클릭해 다운로드합니다.

### 다운로드할 파일

| 운영체제 | 파일명 | 설명 |
|---------|------|------|
| Windows | `옵시디언.AI.비서_3.0.0_x64-setup.exe` | 설치 프로그램 (권장) |
| Windows | `옵시디언.AI.비서_3.0.0_x64_en-US.msi` | MSI 패키지 (기업 환경) |
| macOS | `옵시디언.AI.비서_3.0.0_universal.dmg` | Apple Silicon + Intel 통합 |

> **팁:** `.exe` 파일이 `.msi`보다 설치가 더 간편합니다. Windows 사용자는 `.exe`를 권장합니다.

## 3. 설치 — Windows

### 단계별 설치 안내

**1단계:** 다운로드한 `옵시디언.AI.비서_3.0.0_x64-setup.exe` 파일을 **더블클릭**합니다.

**2단계:** "Windows에서 PC를 보호했습니다"라는 파란색 경고창이 뜰 수 있습니다.
- 이것은 코드 서명 인증서가 없어서 나오는 정상적인 경고입니다. 악성 소프트웨어가 아닙니다.
- **"추가 정보"** 링크를 클릭합니다.
- **"실행"** 버튼이 나타나면 클릭합니다.

**3단계:** 설치 마법사가 시작됩니다.
- **"다음(Next)"** 클릭
- 설치 경로는 기본값 그대로 두는 것을 권장합니다.
- **"설치(Install)"** 클릭
- 완료 후 **"마침(Finish)"** 클릭

**4단계:** 설치가 완료되면:
- 시작 메뉴에 **"옵시디언 AI 비서"**가 등록됩니다.
- 바탕화면에 바로가기 아이콘이 자동으로 생성됩니다.
- 둘 중 하나를 더블클릭하면 앱이 실행됩니다.

> **참고:** 설치 경로는 기본적으로 `C:\Users\사용자이름\AppData\Local\옵시디언 AI 비서\` 입니다. 변경하지 않아도 됩니다.

## 4. 설치 — macOS

### 단계별 설치 안내

**1단계:** 다운로드한 `옵시디언.AI.비서_3.0.0_universal.dmg` 파일을 **더블클릭**합니다.
- DMG 파일이 마운트되고 설치 창이 열립니다.

**2단계:** 설치 창에서 **앱 아이콘을 오른쪽 "Applications" 폴더로 드래그**합니다.
- 복사가 완료될 때까지 잠시 기다립니다.

**3단계:** DMG 창을 닫습니다. 바탕화면에 나타난 디스크 이미지 아이콘은 우클릭 → "꺼내기"로 제거할 수 있습니다.

**4단계:** 앱 실행하기:
- **Launchpad** (도크의 로켓 아이콘)에서 "옵시디언 AI 비서"를 찾아 클릭
- 또는 **Finder → Applications 폴더** → "옵시디언 AI 비서" 더블클릭

**5단계 (첫 실행 시 경고):** "확인되지 않은 개발자" 경고가 뜰 수 있습니다.
- 이는 Apple 공증(Notarization) 없이 배포된 앱에 나타나는 정상적인 경고입니다.
- 앱 아이콘에서 **우클릭(Control+클릭)** → **"열기"** 선택
- "확인되지 않은 개발자의 앱입니다. 그래도 열겠습니까?" 대화상자가 나오면 **"열기"** 클릭
- 이후 실행부터는 이 과정 없이 바로 열립니다.

> **주의:** 더블클릭으로는 처음에 열 수 없습니다. 반드시 **우클릭 → 열기** 순서를 따라주세요.

## 5. 첫 실행 — 설정하기

앱을 처음 시작하면 **⚙️ 설정 모달**이 자동으로 열립니다. 아래 항목을 순서대로 입력하세요.

### 계정
| 항목 | 설명 | 예시 |
|-----|-----|-----|
| **본인 확인용 ID** | 노트 저장 시 본인 확인에 사용됩니다. 아무 텍스트나 가능 | `연규`, `yeongyu`, `my-id` |

### API 키
사용할 AI 제공사의 API 키를 입력합니다. 최소 하나는 있어야 합니다. (API 키 받는 방법은 6번 섹션 참조)

| 항목 | 형태 |
|-----|-----|
| **Google Gemini API Key** | `AIza`로 시작 |
| **OpenAI API Key** | `sk-`로 시작 |
| **Anthropic API Key** | `sk-ant-`로 시작 |

### AI 모델
- 기본값인 **Gemini 2.0 Flash**를 권장합니다. 속도와 성능 균형이 가장 좋습니다.
- 더 정확한 답변이 필요하면 **Gemini 2.5 Pro** 또는 **GPT-4o**를 선택하세요.
- 음성 파일 분석은 **Gemini 계열만** 지원합니다.

### 옵시디언 볼트 경로
- **"📁 폴더 선택"** 버튼을 클릭합니다.
- 열리는 파일 탐색기에서 옵시디언 볼트 폴더(Obsidian vault)를 선택합니다.
- 경로가 자동으로 입력됩니다.

### 테마
- **옵시디언 (기본)**: 옵시디언 다크 스타일 (권장)
- **다크**: 순수 다크 테마
- **라이트**: 밝은 테마
- **시스템**: 운영체제 설정 따라감

### 저장
모든 항목을 입력한 뒤 **"저장"** 버튼을 클릭합니다. 설정이 `~/.myai_obsidian_config.json`에 저장됩니다.

> **보안 안내:** 설정 파일에는 API 키가 저장됩니다. 이 파일을 다른 사람과 공유하거나 GitHub에 올리지 마세요. (`.gitignore`에 이미 포함되어 있습니다.)

## 6. API 키 받기

### Google Gemini API 키

1. [Google AI Studio](https://aistudio.google.com/app/apikey) 접속 (구글 계정 필요)
2. **"Create API key"** 버튼 클릭
3. 새 프로젝트 생성 또는 기존 프로젝트 선택
4. 생성된 키(`AIza...`)를 복사
5. 앱 설정 → **Google Gemini API Key** 칸에 붙여넣기

> 무료 티어가 있어 처음 사용해 보기에 적합합니다.

### OpenAI API 키

1. [platform.openai.com](https://platform.openai.com) 접속 (OpenAI 계정 필요)
2. 우측 상단 프로필 → **"API keys"** 클릭
3. **"Create new secret key"** 클릭
4. 키 이름 입력(선택) → **"Create secret key"** 클릭
5. 생성된 키(`sk-...`)를 즉시 복사 (이 창을 닫으면 다시 볼 수 없습니다)
6. 앱 설정 → **OpenAI API Key** 칸에 붙여넣기

> API 사용량에 따라 요금이 부과됩니다. 처음에는 소액($5~10)을 충전해두면 됩니다.

### Anthropic Claude API 키

1. [console.anthropic.com](https://console.anthropic.com) 접속 (Anthropic 계정 필요)
2. 왼쪽 메뉴 → **"API Keys"** 클릭
3. **"Create Key"** 버튼 클릭
4. 키 이름 입력 → **"Create Key"** 클릭
5. 생성된 키(`sk-ant-...`)를 즉시 복사 (이 창을 닫으면 다시 볼 수 없습니다)
6. 앱 설정 → **Anthropic API Key** 칸에 붙여넣기

> 음성 파일 분석은 Claude에서 지원되지 않습니다. 음성 분석에는 Gemini를 사용하세요.

## 7. 사용하기 — 3개 탭 설명

앱 메인 화면에는 3개의 탭이 있습니다.

### 📁 파일/음성/코드 분석 탭

텍스트 파일, 이미지, 코드 파일, 음성 파일을 AI에게 분석시킬 수 있습니다.

**지원 파일 형식:**
- 이미지: `.png`, `.jpg`, `.jpeg`, `.webp`
- 텍스트/코드: `.py`, `.js`, `.ts`, `.md`, `.txt`, `.json`, `.csv`, `.html`, `.css`
- 음성: `.mp3`, `.wav`, `.m4a`, `.ogg`, `.flac` (Gemini 전용 — 전용 버튼 사용)

**사용 방법:**
1. **파일 첨부** 영역에서 파일을 선택합니다. (일반 파일용)
2. 음성 파일은 반드시 **"🎙️ 음성 파일 선택…"** 버튼을 사용합니다. (8번 섹션 참조)
3. **추가 지시사항** 란에 원하는 분석 방향을 입력합니다. (선택사항)
   - 예: "핵심 개념 위주로 정리해주세요"
   - 예: "이 코드의 버그를 찾아주세요"
4. **"✨ 분석 시작"** 버튼을 클릭합니다.
5. AI가 분석한 결과가 미리보기 영역에 표시됩니다.

### 🌐 웹사이트 요약 탭

웹 페이지 URL을 입력하면 내용을 자동으로 추출해 요약합니다.

**사용 방법:**
1. **웹사이트 URL** 란에 요약할 페이지 주소를 입력합니다.
   - 예: `https://www.example.com/article/123`
2. **추가 지시사항** 란에 원하는 방향을 입력합니다. (선택사항)
   - 예: "주요 기술 스택 위주로 정리해주세요"
   - 예: "한국어로 3줄 요약해주세요"
3. **"🌐 추출 및 요약"** 버튼을 클릭합니다.
4. 웹 페이지 텍스트를 가져와 AI가 요약한 결과가 표시됩니다.

> **주의:** 로그인이 필요한 페이지나 JavaScript로만 렌더링되는 페이지는 내용 추출이 안 될 수 있습니다.

### ✍️ 메모 탭

직접 입력한 메모를 AI가 깔끔하게 정리해 줍니다.

**사용 방법:**
1. **메모 제목** 란에 제목을 입력합니다.
   - 예: `2026-04-25 팀 회의`
2. **메모 내용** 란에 정리되지 않은 내용을 자유롭게 입력합니다.
   - 회의록, 아이디어, 공부 내용 등 무엇이든 가능합니다.
3. **추가 지시사항** 란에 원하는 방향을 입력합니다. (선택사항)
   - 예: "액션아이템을 별도 목록으로 정리해주세요"
   - 예: "마크다운 헤딩을 사용해 구조화해주세요"
4. **"✍️ 정리 시작"** 버튼을 클릭합니다.
5. AI가 정리한 결과가 미리보기 영역에 표시됩니다.

## 8. 음성 파일 분석 — 특별 안내

음성 파일 업로드는 **일반 파일 첨부 방법과 다릅니다.** 이 점이 중요합니다.

### 왜 전용 버튼이 따로 있나요?

Tauri 데스크톱 앱은 보안을 위해 웹 브라우저처럼 파일 시스템에 직접 접근하는 것을 제한합니다. 음성 파일은 크기가 크기 때문에 일반 파일 입력창 대신 **네이티브 파일 선택 대화상자**를 사용해야 정확한 파일 경로를 얻을 수 있습니다.

### 사용 방법

1. **"🎙️ 음성 파일 선택…"** 버튼을 클릭합니다.
2. 운영체제의 기본 파일 탐색기가 열립니다.
3. 분석할 음성 파일을 선택합니다.
   - 지원 형식: `.mp3`, `.wav`, `.m4a`, `.ogg`, `.flac`
4. 선택한 파일명이 버튼 옆에 표시됩니다.
5. 추가 지시사항을 입력한 뒤 **"✨ 분석 시작"** 버튼을 누릅니다.

### 지원 AI 모델

| AI 제공사 | 음성 파일 지원 여부 |
|---------|----------------|
| Google Gemini | ✅ 지원 (권장) |
| OpenAI GPT | ✅ 지원 |
| Anthropic Claude | ❌ 지원 안 됨 |

> 음성 분석을 사용하려면 설정에서 모델을 **Gemini** 또는 **GPT** 계열로 선택하세요. Claude 모델이 선택된 상태에서는 음성 분석이 실행되지 않습니다.

## 9. 옵시디언에 저장

AI 분석 결과가 미리보기에 표시된 뒤 아래쪽 **"💾 옵시디언에 저장"** 섹션이 나타납니다.

### 저장 방법

**1단계 — 저장 폴더 선택:**
- 화면 왼쪽 사이드바의 **"저장할 하위 폴더"** 드롭다운에서 저장할 폴더를 선택합니다.
- 새 폴더를 만들려면 **"＋ 새 폴더 만들기"** 섹션을 펼쳐서 폴더 이름을 입력하고 **"생성"** 버튼을 누릅니다.

**2단계 — 본인 확인:**
- **"본인 확인용 ID"** 란에 설정에서 입력한 ID와 동일한 값을 입력합니다.
- 이 값이 다르면 저장이 되지 않습니다. (실수로 덮어쓰는 것을 방지하기 위한 기능입니다)

**3단계 — 파일명 입력:**
- **"파일명"** 란에 저장할 파일 이름을 입력합니다. `.md` 확장자는 자동으로 붙습니다.
- 예: `2026-04-25 팀 회의 정리` → `2026-04-25 팀 회의 정리.md`로 저장됩니다.

**4단계 — 저장:**
- **"✅ 저장"** 버튼을 클릭합니다.
- 성공하면 화면 상단에 저장 완료 알림이 표시됩니다.

### 파일명 중복 처리

같은 폴더에 같은 이름의 파일이 이미 존재하면, 자동으로 번호가 붙어서 저장됩니다.

| 상황 | 저장되는 파일명 |
|-----|-------------|
| 처음 저장 | `회의 정리.md` |
| 같은 이름 두 번째 | `회의 정리_1.md` |
| 같은 이름 세 번째 | `회의 정리_2.md` |

## 10. 업데이트

### 새 버전이 나왔을 때

1. GitHub Releases 페이지에서 새 버전의 설치 파일을 다운로드합니다.
2. **기존 앱을 제거할 필요 없습니다.** 새 설치 파일을 실행하면 자동으로 덮어씌워집니다.
3. 설정(API 키, 옵시디언 경로 등)은 `~/.myai_obsidian_config.json`에 저장되어 있어 업데이트 후에도 유지됩니다.

### 현재 버전 확인

앱 상단 제목 옆에 버전이 표시됩니다: **"옵시디언 AI 비서 v3.0"**

## 11. 제거

### Windows에서 제거

1. **시작 메뉴** → **설정(톱니바퀴 아이콘)** 클릭
2. **앱** → **앱 및 기능** 클릭
3. 목록에서 **"옵시디언 AI 비서"** 찾기
4. 클릭 → **"제거"** 버튼 클릭
5. 확인 창에서 **"제거"** 클릭

> 설정 파일(`~/.myai_obsidian_config.json`)은 제거 후에도 남아있습니다. 완전히 삭제하려면 파일 탐색기에서 직접 삭제하세요.

### macOS에서 제거

1. **Finder** → **Applications(응용 프로그램)** 폴더 열기
2. **"옵시디언 AI 비서"** 앱 아이콘을 **휴지통으로 드래그**
3. 또는 앱 아이콘에서 **우클릭 → "휴지통으로 이동"** 선택

> 설정 파일(`~/.myai_obsidian_config.json`)은 제거 후에도 남아있습니다. 터미널에서 `rm ~/.myai_obsidian_config.json`으로 삭제할 수 있습니다.

## 12. 문제 해결 (자주 묻는 질문 Top 5)

### 문제 1: macOS — "확인되지 않은 개발자" 경고가 뜨고 앱이 열리지 않아요

**원인:** Apple App Store를 거치지 않은 앱은 첫 실행 시 이 경고가 표시됩니다.

**해결 방법:**
1. 앱을 더블클릭하지 말고, 앱 아이콘 위에서 **우클릭(Control + 클릭)**
2. 메뉴에서 **"열기(Open)"** 선택
3. 경고 창이 뜨면 **"열기(Open)"** 버튼 클릭
4. 이후 실행부터는 더블클릭으로 바로 열립니다.

---

### 문제 2: Windows — "Windows에서 PC를 보호했습니다" 경고가 떠요

**원인:** 코드 서명 인증서 없이 배포된 앱에 표시되는 정상적인 경고입니다.

**해결 방법:**
1. 경고 창에서 **"추가 정보"** 텍스트 링크 클릭
2. 아래쪽에 나타나는 **"실행"** 버튼 클릭
3. 설치가 정상적으로 진행됩니다.

---

### 문제 3: API 오류 — "API 키가 유효하지 않습니다" 또는 응답이 없어요

**원인:** API 키가 잘못 입력되었거나 만료되었을 수 있습니다.

**해결 방법:**
1. 오른쪽 상단 **⚙️ 설정** 버튼 클릭
2. 해당 AI 제공사의 API 키 확인
3. 키를 복사해서 다시 붙여넣기
4. API 키 앞뒤에 공백이 없는지 확인
5. 키가 만료된 경우 6번 섹션을 참고해 새 키를 발급받으세요.

---

### 문제 4: 옵시디언 경로가 인식이 안 돼요 / 폴더 목록이 안 떠요

**원인:** 경로가 잘못 입력되었거나, 경로에 접근 권한이 없을 수 있습니다.

**해결 방법:**
1. 오른쪽 상단 **⚙️ 설정** 버튼 클릭
2. 옵시디언 볼트 경로 옆 **"📁 폴더 선택"** 버튼 클릭
3. 파일 탐색기에서 올바른 옵시디언 볼트 폴더를 선택
4. **직접 타이핑하지 말고 반드시 버튼을 사용**하세요. 경로 오타가 없어집니다.
5. **"저장"** 버튼 클릭 후 앱 화면을 확인합니다.

---

### 문제 5: 음성 파일 업로드가 안 돼요

**원인:** 음성 파일은 일반 파일 첨부 버튼이 아닌 전용 버튼을 사용해야 합니다.

**해결 방법:**
1. **"📁 파일/음성/코드 분석"** 탭으로 이동
2. 일반 **"파일 첨부"** 입력창을 사용하지 말고
3. 아래쪽 **"🎙️ 음성 파일 선택…"** 버튼 클릭
4. 파일 탐색기에서 음성 파일 선택
5. AI 모델이 **Gemini** 또는 **GPT** 계열인지 확인 (Claude는 음성 미지원)

---

### 문제 6 (보너스): 앱이 시작조차 안 돼요 (Windows)

**원인:** Microsoft Visual C++ 재배포 가능 패키지가 설치되어 있지 않을 수 있습니다.

**해결 방법:**
1. [Microsoft 공식 사이트](https://aka.ms/vs/17/release/vc_redist.x64.exe)에서 `vc_redist.x64.exe` 다운로드
2. 더블클릭해서 설치
3. 재부팅 후 앱 다시 실행

> Windows 10/11 최신 업데이트가 적용된 환경에서는 이 문제가 거의 없습니다.
